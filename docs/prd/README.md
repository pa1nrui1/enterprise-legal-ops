## 开发口径

- 输出永远是企业内部文件。
- 不生成律师顾问版或正式法律意见。
- 合同审查唯一交付物是 Word 修订格式合同审核稿。
- 飞书只用于日历提醒，不用于合同审查交付。
- 所有本地目录用 `<company_workspace>` 占位，不写死个人路径。
